SpecAI
======

Local development
-----------------

Prerequisites:
- Node.js 20+ (recommended)
- npm (comes with Node.js)

Install dependencies:

1) Backend
	- cd backend
	- npm install

2) Frontend
	- cd frontend
	- npm install

Run in development:

1) Backend (API)
	- cd backend
	- npm run dev

2) Frontend (Web)
	- cd frontend
	- npm run dev

By default:
- Frontend runs on http://localhost:3000

Production build
----------------

Backend:
- cd backend
- npm run build
- npm run start

Frontend:
- cd frontend
- npm run build
- npm run start
